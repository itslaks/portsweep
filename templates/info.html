About Nmap

Nmap (Network Mapper) is a free, open-source tool used to discover hosts and services on a computer network, thus creating a "map" of the network. Features include host discovery, port scanning, version detection, and OS detection.

How Nmap Works in Linux

In Linux, Nmap can be run from the command line. Here are some common commands:

    - `
  • nmap [IP address]: Basic scan
    • ` - `
  • nmap -sT [IP address]: TCP Connect Scan - Performs a full TCP connection to determine if ports are open
    • ` - `
  • nmap -sS [IP address]: SYN Scan - Sends SYN packets to determine if ports are open without completing the handshake
    • ` - `
  • nmap -sU [IP address]: UDP Scan - Checks for open UDP ports
    • ` - `
  • nmap -sA [IP address]: ACK Scan - Sends ACK packets to determine if ports are filtered or unfiltered
    • ` - `
  • nmap -sW [IP address]: Window Scan - Uses TCP window size to determine open ports
    • ` - `
  • nmap -sX [IP address]: Xmas Scan - Sends TCP packets with the FIN, PSH, and URG flags set to determine if ports are open
    • ` - `
  • nmap -sN [IP address]: Null Scan - Sends TCP packets with no flags set to determine if ports are open
    • ` - `
  • nmap -sI [IP address]: Idle Scan - Uses a third-party host to send packets, making the scan appear as if it originated from that host
    • ` - `
  • nmap -sO [IP address]: IP Protocol Scan - Identifies the IP protocols supported by the target
    • ` - `
  • nmap -sSA [IP address]: TCP SYN/ACK Scan - Sends SYN/ACK packets to determine if ports are open
    • ` - `
  • nmap -sF [IP address]: TCP FIN Scan - Sends TCP packets with the FIN flag set to determine if ports are open
    • ` - `
  • nmap -sM [IP address]: TCP Maimon Scan - Sends TCP packets with the FIN/ACK flags set to determine if ports are open
    • ` - `
  • nmap -sAW [IP address]: TCP ACK Window Scan - Uses the TCP window field to determine open ports
    • ` - `
  • nmap -sP [IP address]: Custom TCP Scan - Allows customization of TCP packet flags for scanning
    • ` - `
  • nmap -sV [IP address]: Service Version Scan - Attempts to determine the version of the services running on open ports
    • ` - `
  • nmap -sC [IP address]: Default Script Scan - Runs Nmap's default set of scripts against open ports
    • ` - `
  • nmap -O [IP address]: OS Detection Scan - Attempts to determine the operating system of the target host
    • ` - `
  • nmap --traceroute [IP address]: Trace Route Scan - Performs a traceroute to discover the path packets take to the target
    • ` - `
  • nmap -PT [IP address]: TCP Ping Scan - Sends TCP packets to check if hosts are up
    • ` - `
  • nmap -PE [IP address]: ICMP Ping Scan - Sends ICMP Echo Request packets to check if hosts are up
    • ` - `
  • nmap -PP [IP address]: Timestamp Ping Scan - Sends ICMP Timestamp Request packets to check if hosts are up
    • ` - `
  • nmap -PA [IP address]: TCP ACK Ping Scan - Sends TCP ACK packets to check if hosts are up
    • ` - `
  • nmap -PU [IP address]: UDP Ping Scan - Sends UDP packets to check if hosts are up
    • ` - `
  • nmap -PS [IP address]: TCP SYN Ping Scan - Sends TCP SYN packets to check if hosts are up
    • ` - `
  • nmap -PO [IP address]: IP Protocol Ping Scan - Sends IP packets with a specific protocol to check if hosts are up
    • ` - `
  • nmap --broadcast [IP address]: Broadcast Ping Scan - Sends broadcast packets to discover hosts on a local network
    • ` - `
  • nmap -sL [IP address]: List Scan - Lists the targets to scan without actually scanning them
    • ` - `
  • nmap -sS -p 53 [IP address]: DNS Scan - Performs a SYN scan specifically on the DNS port (53)
    • ` - `
  • nmap -f [IP address]: Fragmentation Scan - Sends fragmented IP packets to evade packet filters
    • ` - `
  • nmap -6 [IP address]: IPv6 Scan - Performs a scan on IPv6 addresses
    • ` - `
  • nmap -D [IP address]: Decoy Scan - Uses decoy addresses to obscure the source of the scan
    • ` - `
  • nmap -S [IP address]: Spoofed Scan - Spoofs the source IP address of the scan packets
    • ` - `
  • nmap -b [IP address]: FTP Bounce Scan - Uses an FTP server to bounce scan packets to the target
    • ` - `
  • nmap --scanflags [IP address]: Custom Packet Scan - Customizes the TCP packet flags for the scan
    • ` - `
  • nmap --data-length [IP address]: Data Length Scan - Sets the length of the data portion of the packets
    • ` - `
  • nmap --ttl [IP address]: Time To Live (TTL) Scan - Sets the IP Time To Live (TTL) field of the packets
    • ` - `
  • nmap --cookie-echo [IP address]: Cookie Echo Scan - Uses TCP Cookie Transactions to detect open ports
    • ` - `
  • nmap --scan-delay [IP address]: Scan Delay - Sets the delay between scan probes
    • ` - `
  • nmap --max-parallelism [IP address]: Max Parallelism - Sets the maximum number of parallel operations
    • ` - `
  • nmap --max-rtt-timeout [IP address]: Max RTT Timeout - Sets the maximum round-trip time (RTT) for scan probes
    • ` - `
  • nmap --min-rtt-timeout [IP address]: Min RTT Timeout - Sets the minimum round-trip time (RTT) for scan probes
    • ` - `
  • nmap --initial-rtt-timeout [IP address]: Initial RTT Timeout - Sets the initial round-trip time (RTT) for scan probes
    • ` - `
  • nmap --host-timeout [IP address]: Host Timeout - Sets the maximum time to scan a single host
    • ` - `
  • nmap --max-retries [IP address]: Max Retries - Sets the maximum number of retransmissions
    • ` - `
  • nmap --min-hostgroup [IP address]: Min Hostgroup - Sets the minimum number of hosts in a group for parallel scanning
    • ` - `
  • nmap --max-hostgroup [IP address]: Max Hostgroup - Sets the maximum number of hosts in a group for parallel scanning
    • ` - `
  • nmap --min-rate [IP address]: Min Rate - Sets the minimum number of packets per second
    • ` - `
  • nmap --max-rate [IP address]: Max Rate - Sets the maximum number of packets per second
    • ` - `
  • nmap --max-targets [IP address]: Max Targets - Sets the maximum number of targets to scan
    • ` - `
  • nmap --max-hosts [IP address]: Max Hosts - Sets the maximum number of hosts to scan
    • `
